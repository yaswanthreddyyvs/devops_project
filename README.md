# Predictive Maintenance System

## Overview
Predictive maintenance ML app to predict machine failure using operational parameters.

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Docker
- AWS EC2
- GitHub

## Features
- Failure prediction
- Confidence score
- Feature engineering
- Real-time prediction

## Deployment
Dockerized and deployed on AWS EC2.

## Run locally

docker build -t predictive-maintenance .
docker run -p 8501:8501 predictive-maintenance
